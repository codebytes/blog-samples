# Blog Samples

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Companion code samples for blog posts by [Chris Ayers](https://chris-ayers.com/). Each post walks through real-world scenarios with working code you can clone, run, and adapt for your own projects.

Browse the blog at [chris-ayers.com](https://chris-ayers.com/) for in-depth articles on .NET, Azure, DevOps, AI, and distributed systems.

## Sample Collections

Each subdirectory contains a self-contained sample with its own README, prerequisites, and getting started instructions.

| Collection | Description |
|------------|-------------|
| [aspire-cli](aspire-cli/) | .NET Aspire CLI series - building, deploying, and debugging distributed applications |
